About

The uCracker is a very simple Java program that sniffs the network and displays by console the IDs and passwords of all of those that try to login to some sites in that same network.
It relies on the fact that the text that a common user puts in one of those sites, it is delivered to the network just as it's been written, so I just my listen to the network (sniff) and display a password when it's detected.



Get started

Once yo have installed the application, run 

~$ java -jar uCracker.jar [-i interface] [-l loggingLevel]

The -i parameter indicates what interface you want to sniff to. We suggest you to spescify one, because if not the program will search for one and there is no guarantee that interface is enabled, or listening passwords. If you dont't now what interface to sniff, run the ipconfig (on Windows) or ifconfig (on any UNIX plataform such as Mac or Linux) command on any shell and it will list all of your interfaces. Then choose one.

Some tips for choosing an interface:
  • If you are on a wireless network, such as WiFi, choose that interface that describes itself as wireless.
  • Choose that interface that is activated.
IMAGE SHOWING IPCONFIG OR IFCONFIG AND CIRCEL ON THOSE THINGS

The parameter -l indicates the logging level of the application and we suggest you not to indicated and so default will be taken. The possible values for logging are

  • TRACE
  • ERROR (default)
  • OFF

If everything went well, you will see something like this

~$ java -jar uCracker.jar --itba --utn --up -i en1
